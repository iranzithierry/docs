## What is ProStack?

ProStack is a powerful Next.js boilerplate designed to accelerate your web application development. It comes pre-configured with essential features, best practices, and developer tools that help you build production-ready applications in less time.

By using ProStack, you can focus on building your unique features instead of setting up the foundation and integrating common functionalities.

## Why Choose ProStack?

Here's why ProStack stands out as the ideal starting point for your next project:

### Production-Ready

ProStack is built with production in mind, implementing best practices for performance, security, and code organization.

### Developer Experience

We prioritize developer experience with a streamlined workflow, comprehensive documentation, and tools that make development efficient and enjoyable.

### Modern Stack

ProStack uses the latest technologies and patterns, ensuring your application is built on a solid, future-proof foundation.

### Comprehensive Features

From authentication to AI integration, ProStack includes all the essential features you need to get your project off the ground quickly.

## Key Features

ProStack includes the following features and technologies:

- **Frontend**: Next.js 15 with App Router
- **Styling**: Tailwind CSS with JustD component library
- **Database**: PostgreSQL with Prisma ORM
- **Authentication**: BetterAuth - a comprehensive authentication system with JWT and OAuth support
- **AI Integration**: OpenAI API integration for AI features
- **Development**: TypeScript for type safety and better developer experience
- **Deployment**: Docker support for easy deployment
